# Day 05 - Leap Year Checker

## Description
Checks whether a given year is a leap year.

## Logic
- Divisible by 400 → Leap Year
- Divisible by 4 but not 100 → Leap Year
- Otherwise → Not a Leap Year

## Sample Output

```
Enter a year: 2024
2024 is a Leap Year.
```

```
Enter a year: 2025
2025 is Not a Leap Year.
```
